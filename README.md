# AI Movie Insight Builder 🎬

A full-stack web application where users enter an IMDb movie ID and get movie details along with AI-based sentiment analysis.

## Live Demo

Frontend:
https://movie-insight-frontend.vercel.app

Backend:
https://movie-insight-backend-production.up.railway.app

Example:
https://your-backend/api/movie/tt0133093

## Tech Stack

Frontend:
Next.js
Tailwind CSS

Backend:
Node.js
Express.js

API:
OMDb API

Deployment:
Frontend → Vercel
Backend → Railway

## Features

- Search movie using IMDb ID
- Display poster, cast, rating, and plot
- AI-generated audience sentiment summary
- Sentiment classification (Positive / Mixed / Negative)
- Input validation
- Loading spinner and error handling
- Responsive UI

## Local Setup

Backend

cd backend  
npm install  
npm run dev

Frontend

cd movie-insight-frontend  
npm install  
npm run dev

Open http://localhost:3000
